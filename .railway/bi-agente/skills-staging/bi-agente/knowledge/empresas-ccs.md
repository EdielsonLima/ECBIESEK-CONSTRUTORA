## Como interpretar

- **codigo** = `id_sienge_centrocusto` — mostrar ao usuário como "código CC"
- **id_interno** = `id_interno_centrocusto` — usar em queries SQL
- **Ativo S** = situação=2 (em operação); N = cancelado/encerrado

## Nota de atualizacao

Query para atualizar:
```sql
SELECT cc.id_sienge_centrocusto, cc.id_interno_centrocusto, cc.nome_centrocusto, cc.situacao_centrocusto, cc.nome_empresa
FROM dim_centrocusto cc ORDER BY cc.nome_empresa, cc.id_sienge_centrocusto;
```
